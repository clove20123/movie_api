My topMovies Documentation

Objective

To build a movies API that will allow users to access information about movies, directors, and genres. Users will be able to sign up and create a favorites list.

URL endpoints

Request URL Method Body data Response
Register User /users POST A JSON object holding data about the user to register A JSON object containing data about the user that was registered
Remove User /users/:Username DELETE None A message indicating the user was removed
Get favorite movies list /users/:Username/favorites GET None A JSON object containing data about the list of favorite movies
Add a movie to the list of favorites /users/:Username/favorites POST A JSON object holding data about the movie to add to the favorites list A JSON object containing data about the movie to add to the favorites lsit
Remove a movie from the favorites list /users/:Username/favorites/:movieID DELETE None A message indicating the movie was removed
Get a list of movies /movies GET None A JSON object containing data about all the movies
Get a list of movies by title /movies/:Title GET None A JSON object containing data about the movie that was selected
Get a list of directors /directors GET None A JSON object containing data about all the directors
Get a list of Directors by name directors/:Name GET None A JSON object containing data about the director selected
Get a list of genres /genres GET None A JSON object containing data about all the genres
Allow the user to login /login POST None A JSON object containing data about the logged in user
